# Release Notes for v15.4.0-canary.15 - April 30, 2025

## Summary

- A number of updates and enhancements have been introduced for Turbopack, adding further efficiency and performance improvements.
- Prettier has now been enabled to streamline code formatting and linting within the project.
- The project has seen significant updates to dependencies, most notably an upgrade to React.
- Bugs affecting certain features, such as the 'x-matched-path' header and svg extension checking, have been fixed.
- Some non-essential dependencies have been dropped from the project to reduce complexity.
